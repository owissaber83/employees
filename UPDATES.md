# 📊 تحديثات نظام تتبع الحضور والانصراف

## ✨ الميزات الجديدة المضافة

### 1️⃣ بحث متقدم عن الحضور والانصراف
- **البحث بنطاق تاريخي**: يمكنك الآن البحث عن سجلات الحضور من تاريخ إلى تاريخ (بدلاً من تاريخ واحد)
- **قائمة موظفين خالية من التكرار**: عند اختيار موظف، يظهر مرة واحدة فقط (بغض النظر عن عدد مرات حضوره)
- **عرض جميع السجلات**: عند اختيار موظف ونطاق تاريخي، يتم عرض جميع سجلات الحضور والانصراف

### 2️⃣ جدول نتائج البحث
يعرض الجدول المعلومات التالية لكل سجل:
- 👤 اسم الموظف
- 📅 التاريخ
- 🕐 وقت الدخول (الحضور)
- 🕕 وقت الخروج (الانصراف)
- ⏱️ عدد الساعات المشتغلة
- ✅/❌ حالة السجل (حاضر أو غائب)

### 3️⃣ إجماليات الحضور والغياب
يتم عرض ملخص شامل يتضمن:
- ✅ **عدد أيام الحضور**: عدد الأيام التي حضر فيها الموظف
- ❌ **عدد أيام الغياب**: عدد الأيام التي غاب فيها الموظف
- 📊 **إجمالي الأيام**: إجمالي الأيام في النطاق المختار
- ⏱️ **إجمالي الساعات**: إجمالي الساعات المشتغلة

### 4️⃣ حساب الراتب المستحق
عند توفر بيانات الموظف، يتم عرض:
- 💰 **الراتب الأساسي**: الراتب الشهري للموظف
- ✂️ **خصم الغياب**: يتم حساب مبلغ الخصم بناءً على:
  - عدد أيام الغياب
  - الراتب اليومي (الراتب الشهري ÷ 26 يوم عمل)
- 💵 **الراتب المستحق**: الراتب الأساسي - خصم الغياب

### 5️⃣ تقرير الراتب الشامل
زر "تقرير الراتب" يفتح تقرير مفصل يتضمن:
- بيانات الموظف الكاملة (الاسم، الوظيفة، القسم، إلخ)
- ملخص الحضور والانصراف
- حساب الراتب مع التفاصيل:
  - الراتب الأساسي
  - البدلات والمكافآت
  - إجمالي الراتب قبل الخصومات
  - الخصومات (غياب، تأمينات، إلخ)
  - الراتب الصافي
  - **الراتب المستحق بعد خصم الغياب**
- سجل تفصيلي بجميع أيام الحضور والانصراف
- إمكانية طباعة التقرير مباشرة

## 🔧 الدوال المضافة

### `fillAdvancedSearchEmpList()`
- تملأ قائمة الموظفين للبحث المتقدم
- تضمن عدم تكرار أسماء الموظفين
- يتم استدعاؤها تلقائياً عند تحميل صفحة الحضور وعند تحديث البيانات

### `renderAdvancedAtSearch()`
- تعرض نتائج البحث المتقدم
- تحسب الإجماليات (أيام الحضور، الغياب، الساعات)
- تعرض الملخص والراتب المستحق (إن توفر)
- يتم استدعاؤها عند اختيار موظف أو تغيير نطاق التاريخ

### `atResetAdvancedSearch()`
- تعيد تعيين جميع فلاتر البحث
- تمسح النتائج والملخصات

### `generateSalaryReport()`
- توليد تقرير الراتب الشامل
- حساب الخصومات والبدلات
- فتح نافذة جديدة بالتقرير يمكن طباعتها

## 📐 معادلات الحساب

### حساب الراتب اليومي:
```
الراتب اليومي = الراتب الشهري ÷ 26 يوم عمل
```

### حساب خصم الغياب:
```
خصم الغياب = عدد أيام الغياب × الراتب اليومي
```

### الراتب المستحق:
```
الراتب المستحق = الراتب الصافي - خصم الغياب
```
